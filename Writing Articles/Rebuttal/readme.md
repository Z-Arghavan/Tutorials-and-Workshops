# Rebuttal Template

This folder contains a LaTeX template for writing responses to reviewers. 
I use Overleaf: https://www.overleaf.com/project


## How to use
1. Open `MyRebuttalTemplate.tex`
2. Replace the paper title
3. Paste reviewer comments into `point`
4. Write your responses in `reply`
5. Compile with LaTeX or use Overleaf
---

Feel free to adapt it to your writing style or journal requirements.
